簡介：

以Java製作一個單機版飛機空中大戰遊戲，敵我雙方都有飛機可以射擊砲彈，敵方並有大魔王；

必須躲過敵方飛機及大魔王的砲彈，並且以砲彈攻擊大魔王，會計算得分。

若大魔王的血條歸零則破關成功，自己的血條歸零則破關失敗。

並可按空白鍵暫停或恢復遊戲，以便遊戲玩家隨時想暫停或繼續玩。
